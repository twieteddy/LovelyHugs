# LovelyHugs
Hug other players with /hug

## Requirements
* Spigot 1.12.2
* Java 1.8

### Permissions
| Permission                 | Description   | Default |
| -------------------------- | ------------- | :-----: |
| lovelyhug.hug | Allow player to hug  | op |
